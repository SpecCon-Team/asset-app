## How to View Your Data

### Option 1: Using the UI (Recommended)
1. Open your application
2. Login with your user account
3. Navigate to "My PEG" page
4. You'll see the interactive map with all provinces
5. Click on any province to see the clients
6. The client count badges will show on the map

### Option 2: Using the Database Query
```bash
# View all clients
node -e "const {PrismaClient} = require('@prisma/client'); const p = new PrismaClient(); p.pEGClient.findMany().then(c => console.log(c)).then(() => p.\$disconnect())"
```

### Option 3: Load Sample Data Button
- In the "My PEG" page, click the green "Load Sample Data" button
- This will load all 20 sample clients across all provinces
- Data is saved to the database permanently
